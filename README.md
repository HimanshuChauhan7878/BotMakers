# 🤖 BotLeague – India's Ultimate Robotics Arena

BotLeague is a modern, responsive landing page designed for a national robotics competition platform. The website showcases robotics events, competitions, rankings, categories, disciplines, sponsors, and community engagement opportunities for robotics enthusiasts across India.

---

## 🚀 Features

* **Modern Dark-Themed UI**
* **Responsive Design** for desktop and mobile devices
* **Sticky Navigation Bar**
* **Hero Section** with live event highlight
* **Competitions & Events Dashboard**

  * Live Competitions
  * Upcoming Events
  * Past Results
* **User Journey Visualization**
* **About BotLeague Section**
* **Competition Categories**
* **Competition Disciplines Showcase**
* **League Advantages**
* **Community Registration Forms**
* **Sponsors Section**
* **Social Media Footer**

## 📱 Responsive Design

The website automatically adjusts for smaller screens using CSS media queries.

```css
@media(max-width:980px)
```

Features:

* Single-column layout
* Hidden navigation menu
* Optimized spacing
* Mobile-friendly cards

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/botleague.git
```

2. Navigate to the project folder

```bash
cd botleague
```

3. Open `index.html` in your browser

```bash
start index.html
```

or simply double-click the file.
